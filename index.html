1. História da Web

Entender um pouco da história dos computadores, o surgimento da internet e compreender como funciona toda a comunicação quando você acessa uma página web ou utiliza um aplicativo.

Primeiros Computadores

1ª Geração (1944 a 1959)

  • Harvard Mark I (1944): Um dos primeiros computadores eletromecânicos programáveis, desenvolvido pela Universidade de Harvard em colaboração com a IBM. Realizava cálculos automáticos usando relés e interruptores e foi utilizado para cálculos complexos durante a Segunda Guerra Mundial.
  • Colossus (1946): Computador desenvolvido para cálculos e análises criptográficas, criado pela Alemanha. Foi de grande importância para cálculos relacionados à Segunda Guerra Mundial.
  • ENIAC (1946): Um dos primeiros computadores eletrônicos de propósito geral, mais automatizado que o Mark I. Foi nesta época que surgiu o termo "bug", pois as máquinas esquentavam muito e atraíam insetos, que causavam falhas no equipamento.

2ª Geração (1959 a 1965)

A evolução das válvulas para os transistores tornou os computadores menores e mais eficientes.

3ª Geração (1965 a 1970)

Introdução dos circuitos integrados, que incluíam transistores, capacitores e resistores em um único chip, permitindo maior miniaturização e aumento de potência dos computadores.

4ª Geração (a partir de 1971)

Caracteriza-se pelo uso de microprocessadores, a contínua miniaturização dos componentes eletrônicos e o desenvolvimento dos computadores pessoais que conhecemos hoje.

5ª Geração

Envolve o desenvolvimento de tecnologias avançadas como inteligência artificial (IA), computação em nuvem, metaverso e outras inovações que continuam a evoluir.

2. O que são Clients?

Clients, ou clientes, são dispositivos ou programas que solicitam serviços ou recursos de um servidor. No contexto da web, o client é geralmente o navegador de internet usado para acessar sites e serviços online. Quando você digita um endereço de website e pressiona Enter, o seu navegador (client) envia uma solicitação ao servidor onde o site está hospedado.

Os clients podem ser variáveis, incluindo navegadores web, aplicativos móveis, e até mesmo dispositivos IoT (Internet das Coisas) que interagem com a internet.

3. O que são Servers?

Servers, ou servidores, são sistemas de hardware ou software que fornecem serviços, dados ou recursos a outros dispositivos ou programas, conhecidos como clients. No contexto da web, um servidor armazena, processa e responde às solicitações feitas por clientes, como navegadores de internet.

Os servidores podem ser responsáveis por hospedar sites, gerenciar bancos de dados, fornecer e-mails, e muito mais. Eles geralmente operam continuamente e podem ser acessados a partir de diferentes locais através da rede.

4. Sobre linguagens de programação web

As linguagens de programação web são usadas para criar e gerenciar conteúdos e funcionalidades de websites e aplicativos web. Elas podem ser divididas em duas categorias principais:

  • Linguagens Client-Side: Estas linguagens são executadas no navegador do usuário e controlam a apresentação e interação da página. Exemplos incluem HTML, CSS e JavaScript.
  • Linguagens Server-Side: Estas linguagens são executadas no servidor e lidam com a lógica de negócios, manipulação de dados e interações com bancos de dados. Exemplos incluem PHP, Python, Ruby, e Node.js.

HTML (HyperText Markup Language)
É uma linguagem de marcação usada para estruturar conteúdo na web, mas não é considerada uma linguagem de programação porque não possui lógica de controle ou manipulação de dados. CSS (Cascading Style Sheets) é usado para estilizar o conteúdo HTML, enquanto JavaScript permite a criação de interatividade e dinamicidade nas páginas web.

Já no lado do servidor, linguagens como PHP e Python permitem que os desenvolvedores criem aplicações web robustas e interativas, gerenciem dados e interajam com bancos de dados.

Pioneiros da Web

Grace Murray Hopper

  • Pioneira na programação de computadores.
  • Desenvolvimento do primeiro compilador de linguagem de programação.
  • Contribuições para a criação de linguagens de programação como COBOL.

Joseph Carl Robnett Licklider

  • Visão de uma "Rede Intergalática de Computadores".
  • Influência sobre a criação da ARPANET.
  • Contribuições para o conceito de computação interativa.

Robert E. Kahn

  • Co-criador do protocolo TCP/IP.
  • Trabalho essencial para a arquitetura da Internet.
  • Envolvimento no desenvolvimento do DARPA Internet Program.

Tim Berners-Lee ****

  • Inventor da World Wide Web.
  • Desenvolvimento do primeiro navegador e servidor web.
  • Criação do protocolo HTTP e do sistema de URLs.

Marc Andreessen

  • Co-criador do Mosaic, um dos primeiros navegadores gráficos.
  • Co-fundador da Netscape Communications.
  • Influência no desenvolvimento inicial da web comercial.

Bill Gates

  • Fundador da Microsoft.
  • Papel na popularização dos PCs e do sistema operacional Windows.
  • Estratégias para a entrada da Microsoft no mercado de navegadores com o Internet Explorer.

Steve Jobs

  • Co-fundador da Apple.
  • Inovações na interface gráfica de usuário.
  • Contribuições para o desenvolvimento de produtos que transformaram o acesso e uso da Internet, como o iPhone e o iPad.
